# Vikas - Personal Website

Personal portfolio built with [React](https://react.dev) featuring animated particles, dark/light theme toggle, and responsive design.

## Getting Started

1. **Clone the repo:**
   ```sh
   git clone https://github.com/VikasKushwaha5424/vikas-personal-website.git
   ```

2. **Navigate into the directory:**
   ```sh
   cd vikas-personal-website
   ```

3. **Install dependencies:**
   ```sh
   npm install
   ```

4. **Start the development server:**
   ```sh
   npm start
   ```

5. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build for Production

```sh
npm run build
```
